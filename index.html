Creating A Repository And Clone

    create

  • log on to github and select the plus (+) sign in. My GitHub
  • Fill out page to create the Repository. (licence(mit))(gitignore-node) check(public)(readme)Then click create.

    • clone

  • Make sure you are in Repository then select green code button.
  • Click SSH button when in code button then copy url.
  • Go to command center. Select the [cd] directory you want the clone in.
  • Once in the [cd] directory you want to be in type [git clone] space and url you copied enter all finished.

    • Check

  • Check that it worked enter the [cd] name of Repository.Then enter command [git remote -v] to see 2 files with name one for(push) other(fetch).

Command Codes

  • [ls] check status lists the contents of your current working directory
  • [cd "directory name"] change directory
  • [pwd] to check the path of the file
  • [cd ..] to go back one directory
  • [cd] to exit all files
  • [mkdir] create a new directory
  • [touch] create a new file in directory
  • [git status] track the file
  • [git add -A] add all the files that have not been added
  • [git commit -m "give a reference of what you did"] thats to commit the changes to the project
  • [git pull] check for any pushes
  • [git push origin main] to push the updates to the main file
